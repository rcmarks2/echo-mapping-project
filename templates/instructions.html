How to Use the EV & Diesel Route Comparison Tool

Single Route Entry

To compare the EV and diesel route for a single trip:

  1. Go to the home page.
  2. Enter a valid U.S. city and state for both start and destination (e.g., Chicago, IL to Atlanta, GA).
  3. Optionally enter the miles per gallon (MPG) of your diesel truck (defaults to 9 MPG if left blank).
  4. Enter the number of annual trips you expect to make on this route.
  5. Click Compare Routes to view maps, cost, and emissions for both EV and diesel.

Batch Upload (Multiple Routes)

To compare routes across many locations at once:

  1. Prepare an Excel file using the following column headers in row 1:
    • Start City
    • Start State
    • Destination City
    • Destination State
    • MPG (Optional Will Default to 9)
    • Annual Trips (Minimum 1)
  2. Go to the Batch Upload section on the home page.
  3. Upload your Excel file and click Submit.
  4. Once processing is complete, you will receive a downloadable fullbatchresult.xlsx file showing feasibility and mileage for diesel vs. EV routes.

EV Route Optimization Tool

After generating your fullbatchresult.xlsx file, you can use the EV Optimization tool to identify which routes should be prioritized for EV conversion based on potential cost and emissions savings.

To use the tool:

  1. Click EV Route Optimization from the homepage.
  2. Upload your existing fullbatchresult.xlsx file (previously downloaded from the batch upload).
  3. Enter the percentage of routes you'd like to convert to EV (for example, enter 25 to convert the top 25% of routes).
  4. Click Optimize EV Adoption.
  5. You will receive a downloadable file with the selected EV routes highlighted in yellow.
